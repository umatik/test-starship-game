## How to install

Laravel:
1) Clone repo
2) Run command: composer install
3) Run command: php artisan key:generate

VUE:
1) Run command: npm install
2) Run command: npm run dev

MongoDB:
1) Run command: php artisan db:seed

Run project:
1) Run command: php artisan serve
